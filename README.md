# 🚛 Control de Acceso a Proveedores

Aplicación web tipo PWA para gestionar el acceso de proveedores en planta, con control de entradas, salidas, evidencias y seguimiento en tiempo real.

---

## 🚀 Demo
👉 https://TU_USUARIO.github.io/control-proveedores/

---

## ✨ Características

- 🔐 Autenticación de usuarios (Firebase Auth)
- 👥 Control por roles (admin, guardia, calidad)
- 📋 Registro de visitas de proveedores
- 📸 Captura de evidencia (fotos/documentos)
- 📊 Seguimiento de estatus (entrada, salida, dictamen)
- 🔄 Sincronización en tiempo real (Firestore)
- 📱 Instalable como app (PWA)
- 🌐 Funciona offline (Service Worker)

---

## 🧠 Tecnologías utilizadas

- HTML, CSS, JavaScript (Vanilla)
- Firebase:
  - Authentication
  - Firestore
- Service Worker (PWA)
- GitHub Pages (hosting)
